J3GP Pro(n5000主板)问题及解决

咸鱼淘到的n5000主板，问题多多，一部分解决办法

问题一：刷入linux系统开机半小时后，系统死机，按开机键不再管用需要断电再次开机！

目前发现问题可能是此主板的BIOS对于ACPI功能没有优化，最优解为更新bios，由于本人目前没有重新编译bios的能力，找到一个替代解决办法:即通过修改内核启动参数来告诉Linux不要使用BIOS中那部分有缺陷的功能

方法：
打开终端，输入 sudo nano /etc/default/grub 来编辑GRUB配置文件。

找到以 GRUB_CMDLINE_LINUX_DEFAULT= 开头的那一行。

在引号内，添加我们下面将要尝试的参数。例如，原来是 quiet，可以改成 quiet intel_idle.max_cstate=1

按 Ctrl + X 保存，按 Y 确认，再按 Enter 退出。

非常重要的一步：在终端运行 sudo update-grub 来让配置生效。

重启电脑，观察问题是否解决。

测试采用12V1.5A电源仍然会关机。
测试采用12V2A电源1小时11分左右关机。
综上，改启动项目无效。目前没有更好的办法只能看看官方有没有可能更新适配linux的BIOS了（可能性不大）

问题二：linux插入msata硬盘，开不开机（Windows未发现这个问题）

可能是由于启动顺序的问题，由于没有rtc电池，更改BIOS不会保存，所以改了bios也没有用。
目前这个问题不稳定，有时能开机，有时开不开，没找到原因。
解决办法：
目前采用插入usb3.0拓展坞即可正常开机：原理不太清楚，应该可以采用诱骗原理开机

问题三：开机短接点焊盘掉了

采用键盘接口接地及其旁边针脚短接，同样可以开机。

其他

目前安装了Windows server CPU待机功耗3w左右，整机未测。
